# Titulo y Descripción del Proyecto

Casa de Dios en la Roca.

Página web estática de una iglesia cristiana

## Los Usuarios 
Todo visitante puede navegar esta web y ver las distintas secciones.

Secciones:

- Inicio
- Mensaje
- Eventos
- Servicios
- Nosotros
- Contactos
